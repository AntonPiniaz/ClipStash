# Project information

### Rust web service which allows the user to paste and share clipboard content

* **Features**:
1. Link sharing
2. Clip expiration
3. JSON API
4. Password Protected clips

* **Technical Details**:
1. Async web server
2. Template rendering
3. CLI API client
4. SQLite database
5. Background service
6. Multilayered enterprise architecture
7. Fully testable
